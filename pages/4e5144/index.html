2020上半年总结

趁着闲暇时光，总结一下上半年的生活，年初立的很多flag终是没有实现，但生活还要继续，过去留着回忆，生活仍需努力。

# 1月:恐惧

去年研究生考试结束后，漫长的假期就开始了，由于知道自己多半是凉了，所以回家放松一段时间后，就不紧不慢的开始准备了公务员考试。虽然学习比较枯燥无味，但生活还是轻松愉快。突然某一天，群里的朋友提醒大家购买口罩，说出现了病毒，提醒大家注意一些，但我并未在意，以为只是小问题。随着农历新年的临近，节日气息越来越浓烈，就在人们都在准备节日的时候，武汉的疫情突然严重，电视开始播报相关消息，但仍未觉得会传播到东北，直到我们市开始了交通管制，封车封路才发觉事态的严重，大家开始抢购口罩及抗病毒药物，几乎人人都感到了危机，我也陷入了一丝丝恐惧。

# 2月:不安

随着疫情的发展，全国各地都出现了病例，家里也没有逃过，小区开始全面封闭，一个月几乎没有下楼，社区不断的上门统计人员信息，发放消毒药物。每天都非常不安，不断的关注每天的确诊人数，测量自己的体温，有一点咳嗽或不舒服都觉得自己被感染了，真正的做了一次宅男，每天唯一的运动的就是进行消毒，但随着确诊人数的不断增加以及消息的增多，变得越发坦然，只好相信是福不是祸，是祸躲不过。

# 3月:悲伤

随着时间的发展，以及医护科研等人员的不断努力，疫情逐步趋入稳定，疫情进入了常态化管理，可以适当的外出和购物，人们稍微放松了心情。但令我悲伤的是我的感情，最终定格在3月的某一天，虽然有不舍，虽然很难过，但又无可奈何，只好祝她未来安好，平安喜乐。

# 4月:激动

由于临近毕业，我也一直为工作发愁，透了许多简历，最终石沉大海，三月某一天接到了顺丰的电话，邀请我参加顺丰春招的初试，这应该是人生中第一次真正的面试吧，比较紧张也比较激动，在完成面试后不久又来哈进行了现场面试，最终成功得到了offer，今年如初困难的情况下，顺丰给了我机会，人生的第一次工作，充满了感激。

除此之外还有许多事情：

  1. 突然对业余无线电感兴趣，买了书籍以及sdr，学习摩尔斯电码，了解无线电知识，并打算开启无线电操作证书，但由于懒惰，此项工作尚未完成。
  2. 突然大火的动物之森，最终让我踏入了switch的怀抱，咬咬牙购买了游戏机及卡带，开始了属于自己的岛民生活。
  3. 尽管疫情没有结束，但还是冒着危险和朋友相聚，诉说心事，喝酒畅谈，放松心情。

# 5月:奋斗

尽管哈市疫情突然有所紧张，但我们仍然被通知提前来公司报道，办理入职，开始工作。辗转来哈，租房，工作，开始了为了生活的奔波，为了更好的生活而奋斗，工作有苦有泪，但都是值得，都是一种学习，一种磨练，希望坚持下去，作出一番成就。

# 6月:结束

在经过半年的假期时候，最终迎来了结束。结束了大学生活，无法回学校收拾行李，无法与同学相聚，无法参加毕业典礼，一切都采取了比较潮流的线上•云的方式，大学生活就这么匆匆结束了，毕业典礼当天有许多话想说，但由于工作繁忙也就匆匆而过了，大学生活并不完整，但也就这样了。

# 总结

半年的时间已经过去，许多事情已不知何时发生，但都已过去。上半年除了开始工作，似乎没有什么具有纪念意义的事情了，也没有完成什么目标，下半年多半就是为了赚钱，为了生活而努力了，人生十有八九不如意，生活还要继续下去。

（很久没有写博客了，一篇水文终于结束，写一篇文章的确有些耗费时间，下一篇不知何时，因为搬砖生活的确比较疲惫）
